# ClubSphere

ClubSphere is a friendly page to find club and events around you. Meet new people and plan hangouts. Interact with millions of users while having secured user privacy.

https://clubshpere86.netlify.app

- React

- Firebase
- Stripe

- Massive Collection

- Community Driven
- User Privacy
- Secure payments

## Installation ⚙️

Clone the repo:

```bash
git clone https://github.com/glitch86/clubsphere-server.git
cd clubsphere-server
npm install
```

Set up environment variables by creating a `.env` file in the root directory:

```env
MONGODB_USER=your_DB_userName
MONGODB_PASS=your_DB_password
STRIPE_KEY=your_stripe_key
SITE_DOMAIN=https://clubshpere86.netlify.app
```

Run the application:

```bash
nodemon index.js
```
